1. Preparing
	1. Install git
	2. Test your git with [http://gooseberrycreative.com/cmder/](http://gooseberrycreative.com/cmder/) or your cmd
2. Coding
	1. Before you start to code, please pull the latest repository ($ git pull)
	2. Please code in your branch
